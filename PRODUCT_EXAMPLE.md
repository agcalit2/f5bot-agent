# Product

## Name
op.gg

## Description
op.gg is a League of Legends stats tracker and performance analysis platform. It shows champion stats,
tier lists, match history, and live game analysis to help players improve and make better decisions
in champion select.

## Unique Features
- Real-time live game overlay with champion matchup data
- AI-powered champion recommendations based on your rank and playstyle
- Detailed post-game analysis with improvement suggestions
- Multi-region support with global leaderboards
- Mobile app with push notifications for rank changes

- https://mobalytics.gg
- https://www.leagueofgraphs.com
